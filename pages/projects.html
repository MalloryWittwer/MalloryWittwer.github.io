Web app

Interactive crystallography
  • Data viz
  • Python
  • Dash
  • Matplotlib
  • HTML/CSS

A web-based tool to visualize the orientation distribution of a crystallographic property in a face-centered cubic material. The Inverse Pole Figure, which represents the orientation space, is partitioned evenly using a method based on the stereographic projection of a Fibonacci sphere.
